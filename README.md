# Phishing URL Detection Tool

This project is a cybersecurity-focused Python application that detects
whether a URL is phishing or legitimate.

## Features
- Machine learning model trained to classify phishing URLs
- Flask backend API for predictions
- Accepts URL input and returns risk classification

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML / CSS (basic frontend)

## Purpose
This project was built as part of my cybersecurity learning journey to
understand phishing attacks, detection techniques, and backend API security.

## Status
Project is functional and under active improvement.
